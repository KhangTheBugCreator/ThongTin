# Website cá nhân

Website cá nhân được xây dựng bằng HTML và CSS.

## Nội dung

- Giới thiệu bản thân
- Mục tiêu
- Kế hoạch
- Kinh nghiệm
- Liên hệ

## Cấu trúc

- index.html
- muctieu.html
- kehoach.html
- kinhnghiem.html
- lienhe.html
- style.css

├── index.html # Trang chủ - Giới thiệu bản thân & Vị trí nghề nghiệp
├── muctieu.html # Mục tiêu học tập theo mô hình SMART
├── kehoach.html # Lộ trình thực hiện chi tiết đến năm 2028
├── kinhnghiem.html # Kinh nghiệm sử dụng Git và thiết kế UI/UX
├── lienhe.html # Thông tin liên lạc và bản đồ địa chỉ
├── style.css # File định dạng kiểu dáng toàn bộ website
└── img/ # Thư mục chứa hình ảnh, favicon và banner

## Công nghệ sử dụng

- HTML5
- CSS3

## Cách chạy

Mở file `index.html` bằng trình duyệt.

## Ghi chú

Project phục vụ mục đích học tập.
