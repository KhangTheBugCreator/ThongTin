# Website cá nhân

Website cá nhân được xây dựng bằng HTML và CSS.

## Nội dung

- Giới thiệu bản thân
- Mục tiêu
- Kế hoạch
- Kinh nghiệm
- Liên hệ

## Cấu trúc

- index.html
- muctieu.html
- kehoach.html
- kinhnghiem.html
- lienhe.html
- style.css

## Công nghệ sử dụng

- HTML5
- CSS3

## Cách chạy

Mở file `index.html` bằng trình duyệt.

## Ghi chú

Project phục vụ mục đích học tập.
